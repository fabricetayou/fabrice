## 2. Introduction : Pourquoi Tab XP ?

Le développement mobile exige de la clarté et de l'efficacité. **Tab XP** n'est pas qu'un simple gestionnaire d'onglets ; c'est une amélioration structurelle pour l'éditeur Acode. Développé avec précision par **FKB**, ce plugin comble le fossé entre la simplicité du mobile et la puissance des IDE de bureau.

Que vous travailliez sur un petit script ou sur un projet web complexe, Tab XP vous permet de garder le contrôle total sur vos fichiers ouverts grâce à une interface intuitive et des performances optimisées.

---

## 3. Fonctionnalités Phares (L'Exhaustivité)

* **Gestion Avancée des Onglets :** Une barre d'onglets repensée pour une navigation plus naturelle.
* **Aperçus Dynamiques :** Visualisez vos fichiers d'un coup d'œil sans avoir à les ouvrir complètement.
* **Interface Ultra-Fluide :** Optimisé pour ne pas ralentir votre éditeur, même avec des dizaines de fichiers ouverts.
* **Personnalisation Totale :** Adaptez l'apparence de vos onglets selon vos préférences (couleurs, icônes, tailles).
* **Intégration FKB :** Bénéficiez de l'expertise de la *Fabrice Knowledge Base* pour un code propre et des mises à jour régulières.

## 6. Installation Simple (Guide étape par étape)

1. Ouvrez votre application **Acode**.
2. Rendez-vous dans les **Paramètres** > **Plugins**.
3. Recherchez **"Tab XP"**.
4. Cliquez sur **Installer**.
5. Redémarrez l'application et profitez d'une nouvelle dimension de codage.

*Alternative :* Vous pouvez également télécharger le fichier `.zip` directement depuis le lien officiel : `https://acode.app/plugin/acode.plugin.tab_xp`
